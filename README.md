# Broke Geniuses 🎓💼

**Broke Geniuses** is a premium, highly interactive student peer-to-peer marketplace and tutoring platform. It allows students to monetize their skills (e.g., tutoring, resume review, code debugging) while offering buyers an immersive, AI-assisted digital learning experience.

---

## 🌟 Core Features

### 1. Campus Peer Marketplace
* **Dynamic Services Discovery**: Post, edit, and discover tutoring services, study guides, and campus gigs.
* **Smart Booking Flow**: Secure order tracking from booking request to completion.
* **Sentiment-Tagged Reviews**: Leave feedback on completed orders featuring automatic sentiment parsing and meme attachments.
* **Gamified Achievement Badges**: Dynamic visual badges shelf on profile headers (e.g., *Campus Scholar*, *Side Hustler*, *Master Tutor*, *Study Guru*) earned dynamically through marketplace transactions.

### 2. Live Video Classroom & Whiteboard
* **HTML5 Webcam Streams**: Real-time user camera and audio capture directly inside classrooms.
* **Pulsing Peer Simulations**: Remote peer status displays and animated canvas audio waveforms.
* **Interactive Canvas Whiteboard**: Drawing board with custom brush colors, brush sizes, and erasers.
* **Tactile Audio Feedback**: Modulated sound frequencies played dynamically as you draw on the whiteboard.
* **Tutor Sketch Simulation**: Automatically demonstrates whiteboard drawings (e.g., drawing geometry diagrams) to show off real-time classroom collaboration.

### 3. AI Monitor & Revision Packages
* **Consent-Based Monitoring**: Toggleable AI monitoring that transcripts student-tutor dialogues.
* **Keyword-Tailored Materials**: Local text processor parses transcripts to generate:
  * **Detailed Summaries**: Structured bullet-point summaries of the lesson.
  * **Study Cheat Sheets**: Comprehensive reference notes.
  * **Interactive Quizzes**: Multiple-choice card decks.
* **Gamified MCQ Player**: Play revision quizzes with browser-synthesized success/error sounds and final trophy completion screens.
* **Demo-Bypass Paywall**: The study package is locked behind a ₹100 paywall, supporting direct click-to-unlock checkout fallbacks for testing.

### 4. Administrative Dashboard
* **Visual Insights**: Monitor student registrations, service listings growth, and order distribution charts powered by **Chart.js**.

---

## 🛠️ Technology Stack

* **Backend**: Python, Flask, Flask-SQLAlchemy (SQLite)
* **Frontend**: Vanilla HTML5/CSS3 (Glassmorphic theme), JavaScript (Web Audio API for sound generation, Canvas API)
* **Visuals**: Chart.js, Bootstrap 5, Bootstrap Icons
